# Trivia-Project

## Setup:
- requirements: node & mySQL server up and running.
- see 'example.env' and create .env file accordingly.
1. clone this repo
2. npm install
3. cd client
4. npm install
5. cd ..
6. npm run build
7. npx sequelize db:create
8. npx sequelize db:migrate
9. npx sequelize db:seed:all
20. npm start
21. open localhost:3001
